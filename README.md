# OI-wiki-vscode

在 VS Code 中搜索并浏览 OI-Wiki 页面！

## 安装

前往 VS Code 插件市场，搜索 `OI wiki` 即可。

## 使用

使用 `Ctrl+P` 或 `F1` 打开命令面板，输入 `oiwiki`，按下回车键即可进行搜索。
